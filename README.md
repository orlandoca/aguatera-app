# Aguaterapp 💧

**Aguaterapp** es una solución integral de gestión para administradoras de servicios de agua (Aguateras) y Juntas de Saneamiento en Paraguay. Diseñada para digitalizar el ciclo completo operativo, desde el registro de abonados hasta la cobranza inteligente y el control técnico en campo.

## 🚀 Propuesta de Valor
Aguaterapp transforma la gestión tradicional de planillas y cuadernos en un ecosistema digital eficiente, reduciendo la morosidad y eliminando errores humanos en el proceso de facturación y cobranza.

## 📌 Funcionalidades Principales

### 📋 Gestión de Abonados
- Registro detallado de clientes con coordenadas GPS para localización exacta.
- Clasificación de estados: Activo, Cortado, Baja e Inactivo.
- Buscador optimizado por nombre o cédula.

### 💰 Facturación y Cobranzas
- **Generación Masiva:** Creación automática de deudas mensuales para todo el padrón con un solo clic.
- **Pagos Multicanal:** Registro de cobros vía Efectivo, Transferencia, QR o Cheque.
- **Trazabilidad:** Vinculación directa de pagos a deudas específicas y responsable del cobro (Staff).

### 🛠️ Servicio Técnico y Campo
- **Historial de Cortes:** Auditoría completa de cortes y reconexiones de servicio.
- **Personal de Campo:** Registro de responsables y observaciones técnicas por cliente.

## 🛠️ Stack Tecnológico
- **Frontend:** React 19 (Vite) - Interfaz reactiva de alta fidelidad.
- **Estilos:** TailwindCSS 4 - Diseño moderno y ultra-ligero.
- **Backend:** Supabase (PostgreSQL) - Base de datos en tiempo real con seguridad RLS (Row Level Security).
